Title: invariable variation
Year: 2020
Medium: robotic platform, cube, camera, software, electronic circuit
Size: dimensions variable

作品概述:

Description :

恆定中的變化，賦予機械特定行為，因為人的觀看而不變動，而當人不去觀看裝置 時，裝置才會產生一定的變動性。

This installation is endowed with a specific pattern of
mechanical movements, activated by an invariable
variation. It remains still while watched; it shows a
certain variability while ignored.
